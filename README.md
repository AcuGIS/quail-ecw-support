# Quail Additional Formats

This repository contains installers for additional format supports.

## ECW Support

```bash
git clone https://github.com/AcuGIS/quail-formats.git
cd quail-formats
chmod +x ecw-support.sh
./ecw-support.sh
rm -rf quail-formats

```
